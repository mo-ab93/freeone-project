## Dependencies

### Front-End
- NextJs
- Tailwindcss


### Back-End
- Node.js
- Strapi CMS
- PostgreSQL (pg)
- SendGrid
- Cloudinary


## Getting Started
1. Start the back end => npm run dev from backend-strapi
2. Start the front end => npm run dev from frontend-next
